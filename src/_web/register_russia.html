Информация для проживающих в Российской Федерации.

DEF Group предлагает вам возможность зарегистрировать Tonigy по специальной рублевой цене и без наличия пластиковой платежной карты.

Стоимость Tonigy для проживающих на территории Российской Федерации составляет 3 условных единицы, где 1 условная единица в настоящий момент равняется 30 рублям. Таким образом, регистрация Tonigy будет стоить 90 рублей без учета комиссии за перевод денег, которая зависит от используемого способа оплаты.

Данная акция проводится совместно со службой SWRUB (ЗАО "Центр информационных технологий АдаСофт", г. Кострома), которая берет на себя обязанность заниматься выпиской счетов, приемом оплаты и оформлением документов.

Служба SWRUB позволяет использовать различные методы оплаты, такие как: безналичный расчет для юридических лиц и предпринимателей, банковский перевод для физических лиц, почтовый перевод, телеграфный перевод, WebMoney, PayCash. Для более подробных данных прочтите Информацию по способам оплаты.

Для регистрации Tonigy, пожалуйста, посетите форму заказа на сайте службы SWRUB: http://www.swrub.com/order.html?productid=9-1.

Обратите внимание, что регистрационный код будет выслан только после поступления денег, а не после выписки счета. Также следует учесть, что оплата одной лицензии юридическим лицом не позволяет использовать регистрированную копию на всех компьютерах, принадлежащих юридическому лицу. Так же как и физическое лицо, юридическое лицо может использовать одну регистрированную копию только на одном компьютере. Если вы заинтересованы в льготной покупке большого числа лицензий, пожалуйста, свяжитесь с нами.

DEF Group гарантирует, что, зарегистрировав Tonigy один раз, вы остаетесь зарегистрированным пользователем для всех будущих версий Tonigy. Также вы можете общаться со службой поддержки (tonigy@defgroup.com) используя русский язык.
